# Bingo Maker 🎯

Aplicación web para crear y jugar juegos de Bingo en un único dispositivo. Ideal para jugar en una clase online o con familia ó amigos.

## 📖 Características
- Agregar jugadores con iconos seleccionables
- Seleccionar categorías de palabras
- Generar tarjetas de bingo automáticamente
- Jugar y rastrear ganadores

## 🚀 Cómo usar
1. Descarga el proyecto
2. Abre `index.html` en tu navegador
3. Sigue el flujo: Agregar jugadores → Seleccionar palabras → Jugar
4. Disponible para jugar también en GitHub Pages.

## 📁 Estructura del proyecto
- `index.html` - Página de inicio
- `add-players.html` - Agregar jugadores
- `game-settings.html` - Configuración del juego
- `words-selection.html` - Seleccionar categorías de palabras
- `play-bingo.html` - Tablero de juego
- `img/` - Imágenes y iconos
- `styles.css` - Estilos globales
